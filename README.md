🌞 Dual-Axis Solar Tracker for Maximum Power Generation








📌 Project Overview

This project presents the design and simulation of a Dual-Axis Solar Tracking System to maximize photovoltaic (PV) power generation. The system continuously aligns the solar panel with the sun using azimuth (horizontal) and elevation (vertical) movements, ensuring optimal sunlight exposure throughout the day.

The project integrates MATLAB, Simulink, and SolidWorks to perform solar position calculation, dynamic system modeling, and mechanical validation. Simulation results show improved panel alignment and higher energy capture compared to fixed solar systems.
