## 📌 Project Overview

This project presents the design and simulation of a **Dual-Axis Solar Tracking System** to maximize photovoltaic (PV) power generation. The system continuously aligns the solar panel with the sun using **azimuth (horizontal)** and **elevation (vertical)** movements, ensuring optimal sunlight exposure throughout the day.

The project integrates **MATLAB**, **Simulink**, and **SolidWorks** to perform solar position calculation, dynamic system modeling, and mechanical validation. Simulation results show improved panel alignment and higher energy capture compared to fixed solar systems.

## 🧩 Project Components

### 🔢 MATLAB Module

* Calculates solar azimuth & elevation angles
* Uses date, time, and location inputs
* Generates reference tracking angles

### 🔄 Simulink Model

* Simulates dynamic tracking system
* Implements motor control and response
* Ensures accurate panel alignment

### ⚙️ SolidWorks Design

* 3D mechanical model of tracker
* Motion simulation for validation
* Structural and movement feasibility

### ⚡ Control & Actuation

* Dual motors for axis control
* Worm gear mechanism for stability and torque

### 📊 Performance Analysis

* Comparison: Fixed vs Dual-axis system
* Improved alignment and efficiency
* Simulation-based validation
